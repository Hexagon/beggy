Om Beggy

En svensk begagnatmarknad som den borde vara

Beggy är en enkel, gratis begagnatmarknad med öppen källkod. Inspirerad av hur Blocket fungerade förr i tiden - innan alla annonser krävde betalning och gränssnittet blev överbelamrat med reklam.

Öppen källkod

Beggy är helt öppen källkod under MIT-licens. Det betyder att:

  • Du kan se exakt hur tjänsten fungerar
  • Du kan bidra med förbättringar
  • Du kan köra din egen instans
  • Ingen dold spårning eller annonser

Källkoden finns på GitHub.

Teknisk plattform

  • Backend: Deno 2.5 med Oak-ramverket
  • Databas: Supabase (PostgreSQL + Auth + Storage)
  • Hosting: Deno Deploy
  • Frontend: Vanilla JavaScript med Tailwind CSS (via CDN)

Funktioner

  • ✅ Skapa och hantera annonser
  • ✅ Ladda upp bilder (max 5 per annons)
  • ✅ Sök och filtrera på kategori/ort
  • ✅ Enkelt användarkonto
  • ✅ GDPR-kompatibel (radera all din data)
  • ✅ Helt gratis - inga avgifter

Bidra

Vill du bidra till projektet? Skapa en pull request på GitHub eller rapportera buggar via Issues.

Kontakt

Beggy utvecklas av Hexagon. Kontakta oss via GitHub för frågor eller förslag.
