Integritetspolicy (GDPR)

Senast uppdaterad: 2025-01-01

1. Personuppgiftsansvarig

Beggy är ett öppet källkodsprojekt. Som användare av en självhostad instans är operatören av den instansen personuppgiftsansvarig. För den officiella instansen på beggy.se gäller denna policy.

2. Vilka uppgifter samlar vi in?

Vi samlar in följande uppgifter:

  • Vid registrering: E-postadress, namn, telefonnummer (frivilligt), ort (frivilligt)
  • Vid annonsering: Annonsinnehåll, bilder, plats
  • Automatiskt: IP-adress (endast i serverloggar), sessionsinformation

3. Rättslig grund för behandling

Vi behandlar dina personuppgifter baserat på:

  • Avtal: För att tillhandahålla tjänsten (artikel 6.1.b GDPR)
  • Berättigat intresse: För säkerhet och missbruksskydd (artikel 6.1.f GDPR)

4. Hur använder vi uppgifterna?

  • För att skapa och hantera ditt konto
  • För att visa dina annonser för andra användare
  • För att möjliggöra kontakt mellan köpare och säljare
  • För att förhindra missbruk av tjänsten

5. Delning av uppgifter

Vi delar inte dina uppgifter med tredje part, förutom:

  • När det krävs enligt lag (t.ex. vid polisutredning)
  • Information som du själv väljer att publicera i annonser

6. Lagring och radering

  • Kontoinformation lagras så länge kontot är aktivt
  • Annonser raderas automatiskt efter att du tagit bort dem
  • Serverloggar rensas efter 30 dagar

7. Dina rättigheter enligt GDPR

Du har rätt att:

  • Få tillgång till dina personuppgifter
  • Rätta felaktiga uppgifter
  • Radera ditt konto och all tillhörande data ("rätten att bli glömd")
  • Exportera dina uppgifter i maskinläsbart format (JSON)
  • Invända mot viss behandling

8. Så exporterar eller raderar du din data

Du kan exportera eller radera ditt konto och all tillhörande data genom att:

  1. Logga in på ditt konto
  2. Klicka på "Mina annonser"
  3. Scrolla ner till "Kontoinställningar"
  4. Klicka på "Exportera mina data" för att ladda ner alla dina uppgifter i JSON-format
  5. Klicka på "Radera mitt konto" för att permanent radera allt

All din data, inklusive annonser och bilder, raderas permanent och kan inte återställas vid radering.

9. Cookies

Vi använder endast tekniskt nödvändiga cookies:

  • access_token: För att hålla dig inloggad (7 dagar)
  • refresh_token: För att förnya din session (30 dagar)

Vi använder inga spårningscookies, analyticscookies eller tredjepartscookies. Eftersom endast nödvändiga cookies för autentisering används krävs inget samtycke.

10. Säkerhet

Vi skyddar dina uppgifter genom:

  • Krypterade lösenord (SHA-256 hash)
  • HTTPS-kryptering för all trafik
  • Säkra sessionscookies med HttpOnly-flagga

11. Ändringar i policyn

Vid väsentliga ändringar i denna policy meddelar vi användare via e-post eller på webbplatsen.

12. Klagomål

Om du anser att vi behandlar dina uppgifter felaktigt har du rätt att lämna klagomål till Integritetsskyddsmyndigheten (IMY): www.imy.se

13. Kontakt

För frågor om personuppgiftsbehandling, kontakta oss via GitHub: github.com/Hexagon/beggy
